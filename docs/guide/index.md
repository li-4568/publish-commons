# 介绍

Publish Commons 是一个基于 Vue 3 + TypeScript 的企业级组件库，封装了通用的基础组件、满足业务需求的业务组件以及可定制的高级组件。

## 特性

- **基于 Vue 3**：使用 Composition API，更好的逻辑复用和类型推导
- **TypeScript**：完整的类型定义，提供更好的 IDE 支持
- **Ant Design Vue**：基于 Ant Design Vue 4.x 封装，保持一致的设计语言
- **企业级**：涵盖中后台系统常用的组件场景
- **可定制**：丰富的配置项，支持主题定制和二次开发

## 组件概览

| 类别 | 组件 |
|------|------|
| 基础组件 | Button, Input, Select, Checkbox, Radio, Tooltip, Popover |
| 表单组件 | Form, DatePicker, RangePicker, Cascader, TreeSelect, Editor |
| 数据展示 | Table, Tree, Chart |
| 高级组件 | TableSearch, ImageCropper, Notification, FloatButton |

## 技术栈

- Vue 3.3+
- TypeScript 5.0+
- Vite 5.0+
- Ant Design Vue 4.0+
- vxe-table 4.18+（表格组件）
- ECharts 6.0+（图表组件）
- wangEditor 5.0+（富文本编辑器）
